# 智能电烤箱升级 - 安装指南

## 第一步：准备工作

### 1.1 软件准备
- 安装 Arduino IDE (1.8.x 或更高版本)
- 安装 ESP8266开发板支持
  - 文件 > 首选项 > 附加开发板管理器网址: `http://arduino.esp8266.com/stable/package_esp8266com_index.json`
  - 工具 > 开发板 > 开发板管理器 > 搜索 "esp8266" > 安装
- 安装所需库:
  - ESP8266WebServer (已包含在ESP8266核心中)
  - MAX6675库: 工具 > 管理库 > 搜索 "MAX6675" > 安装

### 1.2 硬件准备
- 按照物料清单采购所有组件
- 准备必要的工具
- 确保工作区域安全整洁

## 第二步：电路连接

### 2.1 低压部分连接 (先完成此部分)
1. 将ESP8266插入面包板
2. 连接MAX6675模块:
   - VCC → ESP8266 3.3V
   - GND → ESP8266 GND
   - CLK → D5 (GPIO14)
   - CS → D6 (GPIO12)
   - DO → D7 (GPIO13)
3. 连接蜂鸣器:
   - 正极 → D2 (GPIO4)
   - 负极 → GND
4. 连接LED指示灯:
   - 正极 → D4 (GPIO2)
   - 负极 → GND (通过220Ω电阻)

### 2.2 高压部分连接 (谨慎操作)
1. **断开电烤箱电源**
2. 连接固态继电器:
   - 控制端(+) → D1 (GPIO5)
   - 控制端(-) → GND
   - 负载端串联接入电烤箱加热管电路
3. **重要**: 确保所有高压连接绝缘良好

### 2.3 热电偶安装
1. 将热电偶探头固定在烤箱内部
2. 选择远离直接加热区域的位置
3. 确保线缆不会接触高温部件

## 第三步：软件烧录

### 3.1 代码配置
1. 打开 `SmartOvenController.ino`
2. 修改WiFi配置:
   ```cpp
   const char* ssid = "你的WiFi名称";
   const char* password = "你的WiFi密码";
   ```
3. 检查引脚定义是否与你的连接一致

### 3.2 烧录程序
1. 选择正确的开发板: 工具 > 开发板 > NodeMCU 1.0
2. 选择端口: 工具 > 端口 > 对应的COM端口
3. 点击"上传"按钮
4. 等待上传完成

### 3.3 测试
1. 打开串口监视器 (波特率115200)
2. 观察启动信息
3. 记录显示的IP地址

## 第四步：功能测试

### 4.1 基础功能测试
1. **温度读取测试**: 用手触摸热电偶，观察温度变化
2. **继电器测试**: 通过Web界面控制继电器开关
3. **蜂鸣器测试**: 检查报警功能

### 4.2 Web界面测试
1. 在浏览器中输入ESP8266的IP地址
2. 测试各项控制功能:
   - 设置目标温度
   - 开始/停止加热
   - 查看实时状态

### 4.3 安全测试
1. 检查所有连接是否牢固
2. 验证过热保护功能
3. 测试异常情况处理

## 第五步：最终安装

### 5.1 内部布线
1. 将控制板固定在烤箱内部安全位置
2. 使用扎带整理线缆
3. 确保所有连接远离高温区域

### 5.2 散热处理
1. 为固态继电器安装散热片
2. 涂抹散热硅脂
3. 确保良好通风

### 5.3 安全检查
1. 使用万用表检查绝缘
2. 验证接地连接
3. 测试漏电保护

## 故障排除

### 常见问题

**Q: WiFi连接失败**
A: 检查SSID/密码，确保信号强度足够

**Q: 温度读数异常**
A: 检查热电偶连接，确保没有短路

**Q: 继电器不工作**
A: 检查控制信号，验证继电器规格

**Q: Web界面无法访问**
A: 检查IP地址，确保设备与电脑在同一网络

### 安全提示
- 操作高压电时务必断电
- 使用绝缘工具
- 不要单独操作
- 首次测试时保持观察

## 维护建议

### 定期检查
- 每月检查连接是否松动
- 清洁热电偶探头
- 检查散热片温度

### 软件更新
- 定期检查固件更新
- 备份当前配置
- 测试新功能后再部署
