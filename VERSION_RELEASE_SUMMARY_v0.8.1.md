# 智能电烤箱控制器 v0.8.1 版本发布总结报告

## 版本信息
- **版本号**: v0.8.1
- **发布日期**: 2025年11月10日
- **发布类型**: 正式版本

## 主要更新内容

### 1. 版本号统一更新
- 将所有文件中的版本信息统一更新为v0.8.1
- 更新固件主文件中的版本常量定义
- 确保版本一致性

### 2. 温度校准重置功能完善
- 后端实现完整的/reset_calibration路由接口
- 前端实现真实API调用而非模拟
- 添加完整的测试验证流程

### 3. 界面布局优化
- 删除重复的WiFi配置卡片，解决界面冗余问题
- 重新整理卡片布局，分为"系统管理"和"高级管理"两个功能区
- 系统管理功能区包含：WiFi配置、设备状态、温度校准、设置与帮助
- 高级管理功能区包含：管理员控制（退出登录、修改密码）、系统管理（WiFi配置、重启设备、恢复出厂设置）、固件升级
- 优化CSS样式，提升界面美观度和用户体验

### 4. OTA升级页面优化
- 优化OTA升级页面视觉设计，采用卡片式布局
- 添加图标和颜色区分固件升级和HTML文件升级入口
- 增强设备状态显示和文件预览功能
- 改进进度反馈和用户交互体验
