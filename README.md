# 智能电烤箱控制器

基于ESP8266的智能电烤箱升级方案，使用MAX6675 K型热电偶进行精确温度控制，通过固态继电器控制加热，具备Web远程控制功能。

## 功能特性

- ✅ **精确温度控制**: MAX6675 K型热电偶，0-1024°C测量范围
- ✅ **远程Web控制**: 通过浏览器控制烤箱，实时监控状态
- ✅ **智能温度调节**: PID式温度控制算法
- ✅ **安全报警**: 蜂鸣器报警，温度异常保护
- ✅ **状态指示**: LED状态指示灯
- ✅ **运行计时**: 加热时间统计
- ✅ **移动端友好**: 响应式Web界面

## 硬件要求

### 核心组件
- ESP8266开发板 (NodeMCU/Wemos D1 Mini)
- MAX6675 K型热电偶模块
- 固态继电器 (25A/220V以上)
- K型热电偶探头
- 有源蜂鸣器
- LED指示灯

### 详细清单
见 [物料清单.md](物料清单.md)

## 快速开始

### 1. 环境准备
- 安装 Arduino IDE
- 安装 ESP8266开发板支持
- 安装 MAX6675库

### 2. 电路连接
参考 [电路连接图.md](电路连接图.md) 完成硬件连接

### 3. 软件配置
1. 修改 `SmartOvenController.ino` 中的WiFi配置
2. 上传程序到ESP8266
3. 通过串口监视器获取设备IP

### 4. 使用控制
在浏览器中输入设备IP地址，即可访问控制界面

## 引脚定义

| 功能 | ESP8266引脚 | 说明 |
|------|-------------|------|
| MAX6675 CLK | D5 (GPIO14) | 时钟信号 |
| MAX6675 CS  | D6 (GPIO12) | 片选信号 |
| MAX6675 DO  | D7 (GPIO13) | 数据输出 |
| 继电器控制 | D1 (GPIO5)  | 固态继电器控制 |
| 蜂鸣器 | D2 (GPIO4)  | 报警提示 |
| LED指示灯 | D4 (GPIO2) | 状态指示 |

## 安全特性

### 硬件安全
- 固态继电器过零触发，减少电磁干扰
- 热电偶高温保护设计
- 继电器负载端完全隔离

### 软件保护
- 温度超限自动切断加热
- 看门狗定时器防死机
- 异常状态自动恢复
