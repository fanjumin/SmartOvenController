# 智能电烤箱控制器

基于ESP8266的智能电烤箱升级方案，使用MAX6675 K型热电偶进行精确温度控制，通过固态继电器控制加热，具备Web远程控制功能?

## 功能特?

- ?**精确温度控制**: MAX6675 K型热电偶?-1024°C测量范围
- ?**远程Web控制**: 通过浏览器控制烤箱，实时监控状?
- ?**智能温度调节**: PID式温度控制算?
- ?**安全报警**: 蜂鸣器报警，温度异常保护
- ?**状态指?*: LED状态指示灯
- ?**运行计时**: 加热时间统计
- ?**移动端友?*: 响应式Web界面

|------|-------------|------|
| MAX6675 CLK | D5 (GPIO14) | 时钟信号 |
| MAX6675 CS  | D6 (GPIO12) | 片选信?|
| MAX6675 DO  | D7 (GPIO13) | 数据输出 |
| 继电器控?| D1 (GPIO5)  | 固态继电器控制 |
