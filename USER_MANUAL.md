## 产品简介

智能烤箱控制器是一款基于ESP8266的Wi-Fi智能控制器，专为传统电烤箱设计。通过本控制器，您可以实现对烤箱的远程监控和控制，包括温度调节、定时设置、预设模式等功能。

### 主要特性
- Wi-Fi连接，支持远程控制
- 实时温度监控与调节
- 多种预设烹饪模式
- 定时功能
- OTA无线固件升级
- 用户友好的Web界面
- 安全保护机制

## 安装指南

### 硬件安装
1. 断开烤箱电源
2. 将智能控制器安装在烤箱控制面板附近
3. 连接控制器到烤箱的加热元件和温度传感器
4. 接通电源并确认指示灯正常

### 网络配置
1. 首次上电后，控制器会创建名为"SmartOven_XXXX"的Wi-Fi热点
2. 使用手机或电脑连接该热点
3. 打开浏览器访问 `http://192.168.4.1`
4. 在网络设置页面输入您的Wi-Fi名称和密码
5. 保存设置后控制器将自动连接到您的家庭网络

## 快速开始

### 访问控制界面
1. 确保设备已连接到您的Wi-Fi网络
2. 打开浏览器，在地址栏输入设备IP地址（默认为192.168.16.104）
3. 输入管理员密码登录（默认密码：admin123）

### 基本操作
1. **设置温度**：在主界面拖动温度滑块或直接输入目标温度值
2. **设置时间**：调整定时器分钟数
3. **启动烤箱**：点击"开始"按钮
4. **暂停/取消**：点击"暂停"或"停止"按钮

## 功能详解

### 温度控制
- 温度范围：50°C - 250°C
- 精度：±5°C
- 支持实时温度曲线显示

### 预设模式
1. **解冻模式**：低温循环，适合食物解冻
2. **烘焙模式**：标准烘焙，适用于蛋糕、面包等
3. **烧烤模式**：高温上火加热，用于食物表面着色
4. **热风循环**：风扇辅助均匀加热

### 定时功能
- 可设置1-120分钟定时
- 时间到自动停止加热
- 支持中途修改定时设置

### 状态监控
- 实时显示当前温度
- 显示剩余时间
- 工作状态指示（待机/加热/暂停/完成）

## OTA升级

智能烤箱控制器支持无线固件升级（OTA），无需拆机即可更新功能。

### 升级步骤
1. 访问控制界面的"系统设置"页面
2. 点击"固件升级"选项
3. 选择新的固件文件（.bin格式）
4. 点击"开始升级"
5. 等待升级完成（期间请勿断电）
6. 设备将自动重启并应用新固件

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无法连接Wi-Fi | 密码错误或信号弱 | 重新配置网络设置，确保密码正确 |
| 无法访问控制界面 | IP地址变更或设备离线 | 检查路由器确认设备IP，或重新连接设备热点 |
| 温度不准确 | 传感器位置不当或老化 | 调整传感器位置，必要时更换传感器 |
| 设备无响应 | 系统卡死 | 长按复位键10秒强制重启 |
| 加热异常 | 继电器故障或线路问题 | 检查硬件连接，联系技术支持 |

## 安全注意事项

1. **电气安全**
   - 安装前务必切断电源
   - 请由专业电工进行安装
   - 确保接地良好

2. **使用安全**
   - 请勿在潮湿环境中使用
   - 请勿触摸加热元件
   - 儿童需在成人监护下使用

3. **维护安全**
   - 清洁前必须断电
   - 请勿用水冲洗控制器
   - 定期检查线路连接

4. **紧急情况**
   - 如发现异味或冒烟，请立即断电
   - 发生火灾时使用干粉灭火器
   - 请联系专业技术支持处理复杂故障
