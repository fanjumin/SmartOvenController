# 智能烤箱控制器 v0.8.1

## 版本信息
- 版本号: v0.8.1
- 发布日期: 2025-11-13
- 项目状态: 已完成并测试通过

## 更新内容
- 修复了烘焙控制卡片UI混乱的问题
- 优化了模式切换按钮的样式，使其更清晰直观
- 调整了自动模式和手动模式区域的布局，提高视觉层次感
- 修复了元素之间的间距问题，使整体布局更协调
- 将所有文件中的版本信息统一更新为v0.8.1

## 目录结构
- `src/` - 固件源代码
- `data/` - Web界面文件
- `docs/` - 文档资料

## 使用说明
1. 烧录固件和文件系统到ESP8266设备
2. 通过Web界面配置WiFi网络
3. 访问设备IP地址开始使用
