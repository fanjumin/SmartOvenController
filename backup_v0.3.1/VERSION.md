# 智能电烤箱控制器 - 版本控制

## 版本历史

### v0.3.1 (当前版本)
- 修复Web服务器无法访问问题
  - 在connectToWiFi()函数中添加setupWebServer()调用
  - 在loop()函数中添加非强制门户模式下的webServer.handleClient()处理逻辑
- 新增完整的LED状态指示系统
  - 配网前状态：LED快闪（500ms间隔）
  - 待机状态：LED慢闪（1000ms间隔）
  - 加热中状态：LED常亮
  - 烘焙结束状态：LED快闪10秒后自动切换待机
- 新增蜂鸣器状态提示功能
  - 配置保存提示：短促"滴"声
  - 烘焙开始提示：中长"滴"声
  - 烘焙完成提示：长"滴"声
  - 错误提示：急促"滴滴"声
- 优化状态优先级管理
- 完善视觉和听觉反馈系统

### v0.2.0
- 修复MAX6675温度传感器库兼容性问题
- 实现手动SPI通信协议
- 优化温度读取稳定性

### v0.1.0
- 初始版本
- 实现强制门户（Captive Portal）配网功能
- 实现设备自动发现功能（UDP广播）
- 实现OTA空中升级功能
- 基础的温度控制和硬件接口
- 版本控制系统建立

### 版本命名规则
- 主版本号.次版本号.修订版本号
- 主版本号：重大功能变更
- 次版本号：新功能添加
- 修订版本号：Bug修复和优化

### 升级说明
- 每次OTA升级前请确保网络连接稳定
- 升级过程中请勿断电
- 升级完成后设备会自动重启
