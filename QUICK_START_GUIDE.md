# 智能烤箱控制器快速上手指南

## 第一步：硬件连接
1. 将MAX6675温度传感器连接到ESP8266的以下引脚：
   - VCC -> 3.3V
   - GND -> GND
   - SO -> D6(GPIO12)
   - CS -> D1(GPIO5)
   - SCK -> D5(GPIO14)

2. 将加热控制电路连接到PWM输出引脚D2(GPIO4)

3. 为ESP8266提供稳定的5V电源

## 第二步：烧录固件
### 方法1: 使用esptool (推荐)
```bash
# 烧录固件
esptool.exe --port COM11 write_flash 0x00000 release/firmware.bin

# 烧录文件系统
esptool.exe --port COM11 write_flash 0x300000 release/littlefs.bin
```

### 方法2: 使用PlatformIO
```bash
# 进入项目目录
cd SmartOvenController

# 烧录固件和文件系统
pio run -t upload
pio run -t uploadfs
```

## 第三步：设备配置
1. 设备首次启动会创建WiFi热点 "SmartOven-XXXX"
2. 连接到该热点 (密码: 12345678)
3. 打开浏览器访问 `192.168.4.1`
4. 在WiFi配置页面输入您的WiFi网络信息
5. 点击保存并等待设备重启

## 第四步：正常使用
1. 设备重启后会连接到您配置的WiFi网络
2. 查找设备IP地址的方法：
   - 查看路由器管理界面的已连接设备列表
   - 使用网络扫描工具
3. 在浏览器中访问设备IP地址
4. 输入用户名 `admin` 和密码 `password` 登录

## OTA升级
设备支持通过Web界面进行OTA升级：
1. 登录设备Web界面
2. 进入"设置帮助"页面
3. 点击"OTA升级"标签页
4. 选择要上传的固件(.bin)或文件系统文件
5. 点击相应按钮开始上传
6. 等待上传完成并自动重启

## 常见问题
### 1. 无法访问设备Web界面
- 检查设备是否正确连接到WiFi
- 确认输入的IP地址是否正确
- 尝试使用ping命令测试设备连通性

### 2. 温度显示异常
- 检查MAX6675传感器连接是否牢固
- 确认传感器供电是否正常
- 查看设备日志获取更多信息

### 3. 加热功能不工作
- 检查加热控制电路连接
- 确认PWM输出引脚配置是否正确
- 验证电源供电是否充足
