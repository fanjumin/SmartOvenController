# 电烤箱智能控制系统 - 电路连接图

## 硬件连接说明

### 所需组件
1. ESP8266开发板 (NodeMCU或Wemos D1 Mini)
2. MAX6675 K型热电偶模块
3. 固态继电器 (SSR) - 建议25A以上
4. 有源蜂鸣器
5. K型热电偶传感器
6. 面包板 + 杜邦线
7. 电源适配器 (5V/2A)

### 引脚连接表

| 组件 | ESP8266引脚 | 说明 |
|------|-------------|------|
| MAX6675 CLK | D5 (GPIO14) | 时钟信号 |
| MAX6675 CS  | D6 (GPIO12) | 片选信号 |
| MAX6675 DO  | D7 (GPIO13) | 数据输出 |
| MAX6675 VCC | 3.3V | 电源正极 |
| MAX6675 GND | GND | 电源负极 |
| 固态继电器控制端 | D1 (GPIO5) | 继电器控制信号 |
| 蜂鸣器 | D2 (GPIO4) | 报警信号 |
| LED指示灯 | D4 (GPIO2) | 状态指示 |

### 电源连接
- ESP8266: 5V USB供电
- 固态继电器: 需要外部电源，控制端接ESP8266，负载端接电烤箱加热管
- MAX6675: 3.3V供电

## 安全注意事项

### 高压安全
1. **固态继电器负载端连接电烤箱加热管，涉及220V交流电**
2. 确保所有高压连接绝缘良好
3. 在通电前检查所有接线
4. 使用合适的继电器容量（建议25A以上）

### 热电偶安装
1. 将热电偶探头固定在烤箱内部合适位置
2. 避免直接接触加热管
3. 确保热电偶线缆远离高温区域

### 散热考虑
1. 固态继电器需要良好散热
2. 建议安装散热片
3. 确保通风良好

## 测试步骤

1. **低压测试**: 先不接加热管，测试继电器控制功能
2. **温度测试**: 用打火机加热热电偶，观察温度读数
3. **网络测试**: 连接WiFi，访问Web界面
4. **完整测试**: 接入加热管进行完整功能测试
